# Privacy And Compliance Boundary

This release intentionally excludes operator data.

Do not add:

- wallet directories or seed phrases
- hotkey names, coldkey names, or local wallet paths
- server IP addresses, SSH aliases, PM2 process names, or local usernames
- raw validator chunks or private live observations
- monitoring logs, dashboard scrapes, or ad-hoc audit logs
- private runbooks or agent transcripts

The model artifact and inference code are publishable for Poker44 model transparency.
